# Graphura – Website Conversion Optimizer

## Quick Start
```bash
pip install -r requirements.txt
python app.py
```
Open http://localhost:5000

## Pages
- `/`          → Dashboard (audit table, KPI cards, issues)
- `/predict`   → ML Conversion Predictor (Decision Tree / Logistic Regression)
- `/checklist` → Optimization Checklist & Wireframe Suggestions

## Model Accuracy
- Decision Tree:        ~90%
- Logistic Regression:  ~77%
